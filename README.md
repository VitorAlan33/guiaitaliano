Note que a pronúncia dessas letras geralmente segue a sonoridade da língua de origem da palavra.

Adendo Importante: A Elisão e as Vogais
Diferente do português, no italiano as vogais devem ser sempre nítidas e bem pronunciadas
Um ponto crucial para o iniciante é notar que a contração (L') só acontece no singular
No feminino plural, por exemplo, o artigo Le nunca se contrai, mesmo se a palavra seguinte começar com vogal, como em "le italiane"
Isso ajuda a manter a clareza sobre a quantidade de coisas de que você está falando.
